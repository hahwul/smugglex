## Basic Usage

Run a basic scan on a target URL:

```bash
smugglex https://target.com/
```

Enable verbose output:

```bash
smugglex https://target.com/ -v
```

Save results to JSON:

```bash
smugglex https://target.com/ -o results.json
```
