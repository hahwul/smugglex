Version history and changes for smugglex.

## Releases

### v0.1.0 - First Release

Initial release of smugglex with core functionality:

- HTTP Request Smuggling detection
- Support for CL.TE, TE.CL, TE.TE, H2C, and H2 attack types
- Timing-based vulnerability detection
- Command-line interface
- JSON output support
- Payload export functionality
- Pipeline support for tool integration
- Localhost access exploitation
